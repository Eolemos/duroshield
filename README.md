# Duroshield — site

Redesenho do site da **Duroshield Spray Systems** — engenharia de impermeabilização
e revestimentos de alto desempenho em poliureia, 20 anos de mercado, São Paulo,
execução em território nacional.

## Narrativa

> **Impermeabilizamos sua planta sem parar sua produção.**

Poliureia aplicada a spray sobre a estrutura existente. Sem demolição, sem remoção
de piso, sem linha parada. Material e mão de obra no mesmo contrato — um só
responsável pelo resultado.

Três camadas sustentam a página:

1. **Continuidade** — aplicação sobre o substrato existente, sem parada de linha.
2. **Responsabilidade única** — material e mão de obra no mesmo contrato.
3. **Prova blue-chip** — Petrobras, GRU Airport, Bayer, Heineken, Unilever, Takeda.

## Sistemas

| Linha | Para |
| --- | --- |
| `SPR092` | Onde não pode vazar — poliureia elastomérica, membrana contínua, 325% de alongamento |
| `FLEX` | Onde o sol destrói — elastomérico acrílico refletivo |
| `COOL` | Onde o calor custa caro — isolante térmico com microesferas |
| `CR` | Onde o químico ataca — epóxi modificado, alta resistência química |

## Nota sobre este repositório

Este repositório é **público** e contém apenas o código do site.

O material de diagnóstico, a análise de concorrentes, o espelho do site antigo e o
deck de apresentação ficam **fora do controle de versão** (ver `.gitignore`) —
são material interno de consultoria e incluem logos de terceiros cujo direito de
uso ainda não foi confirmado.
